# 更新日志

本文件记录栖声播放器的主要变更，格式参考 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)。

## [1.0.0] - 2026-04-29

### 新增

- 将应用整合为 `qisheng_player`，新增栖声品牌名、圆形应用图标、Windows Runner 图标和发布包命名。
- 音乐页新增可开关的右侧歌词预览栏，显示当前播放歌曲的封面、歌名、歌手和滚动歌词。
- 音乐页歌词预览状态写入偏好设置，重启后可恢复上次开关状态。
- 详情页统一视觉骨架，艺术家、专辑和详细信息入口不再跳入突兀的旧深色整页容器。
- 主内容区侧栏切页增加淡入淡出过渡，避免页面切换时突然跳变。
- Now Playing 支持沉浸式与专业式布局，保留竖向/横向歌词、翻译显示和歌词控制自动隐藏。
- 增强曲库页面：A-Z/拼音索引、当前播放定位、多选、排序、视图切换和右键更多菜单保持可用。

### 改进

- 统一主界面、底部播放器、侧栏、页面面板和详情页的玻璃拟态视觉语言。
- 右侧歌词预览采用悬浮式布局，不额外套玻璃框，收起动画复用展开动画的反向节奏。
- 文件夹管理迁移到独立页面，同时保留设置页入口。
- 底部播放器优化随机播放、播放模式、音量、桌面歌词和播放队列入口的布局。
- 动态专辑取色仅影响强调色与背景氛围，减少对文字层级和可读性的干扰。
- 默认 `balanced` 效果档降低动画与模糊成本，减少主界面卡顿。

### 修复

- 修复左侧边栏切页时因 `AnimatedSwitcher` 重复持有 `GlobalKey` 导致的卡死问题。
- 修复播放器内残留中文乱码，包括“文件夹”“专辑”“歌单”“当前版本”“ReplayGain”等可见文案。
- 修复同步歌词当前行文案插值错误，正确显示当前逐字歌词与翻译。
- 修复删除歌曲失败时错误信息未正确插值的问题。
- 修复部分旧偏好导致启动页异常、默认排序异常、启动音量 100% 和快捷键冲突的问题。
- 修复元数据中常见 UTF-8 被误按 Latin-1 / Windows-1252 解码造成的中文、日文和韩文乱码。

### 验证

- 已执行 Dart 格式化、Flutter 静态分析、关键 Widget 测试、Windows Debug/Release 构建和发布打包流程验证。
- 发布脚本输出 `Qisheng-Player-v1.0.0-Windows-x64.zip`，安装器输出依赖本机 Inno Setup 6。

## 历史说明

`1.0.0` 是栖声品牌化后的第一个完整版本。旧的 `1.7.x` 预发布记录已合并进本次版本说明，后续版本将继续按 `1.x.y` 维护。
