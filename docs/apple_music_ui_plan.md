# 沉浸式播放器 UI 计划归档

本文档归档早期“Apple Music 风格 UI”探索。当前项目已经收束为栖声自己的视觉体系，不再直接追随 Apple Music 的配色或组件语言。

## 已落地内容

- 主界面改为侧栏、内容面板和底部播放器组成的桌面播放器布局。
- 统一引入玻璃拟态表面、动态专辑取色、可降级的视觉效果档位和更轻的悬浮控件。
- Now Playing 页面拆分为沉浸式与专业式两种展示方式。
- 底部播放器支持播放控制、播放模式、音量、桌面歌词和队列入口。
- 列表项、专辑封面、歌词和页面切换补充了更细腻的 hover、press、fade 与 Hero 过渡。
- 音乐页新增右侧歌词预览，作为栖声 v1.0 的独立功能，不照搬概念图配色。

## 当前设计原则

- 以当前主题系统为主，所有新组件优先使用 `AppSurface`、主题扩展和现有 motion token。
- 默认效果档优先保证流畅，重动画、强模糊和频谱可视化只在视觉增强场景中启用。
- 动态专辑取色只影响氛围和强调色，不破坏文字可读性与表面层级。
- 详情页、列表页、播放页和设置页保持同一套间距、圆角、按钮和面板语言。

## 后续方向

- 持续减少主界面的持续重绘成本。
- 为歌词、封面和播放队列补充更多回归测试。
- 继续完善发布包、安装器和 GitHub Release 自动化。
