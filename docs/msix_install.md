# MSIX 安装指南

当前 `qisheng_player v1.0.0` 推荐使用 zip 便携包或 Inno Setup 安装器。本文档保留给历史 MSIX 包使用。

如果你下载的是 `.msix` 安装包，请按以下步骤导入证书：

1. 右键 `.msix` 文件，进入“属性”。
2. 打开“数字签名”标签页。
3. 选中签名列表中的证书，点击“详细信息”。
4. 在弹出的窗口中点击“查看证书”。
5. 点击“安装证书”。
6. 选择“本地计算机”，然后选择“将所有的证书都放入下列存储”。
7. 点击“浏览”，选择“受信任的根证书颁发机构”。
8. 完成证书导入后，双击 `.msix` 安装。

证书通常只需要安装一次，后续同一签名的 MSIX 更新可直接安装。
