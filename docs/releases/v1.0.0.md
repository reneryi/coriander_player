# Qisheng Player v1.0

发布日期：2026-04-29

这是栖声品牌化后的第一个完整版本，应用包名与可执行文件已切换为 `qisheng_player`。

## 亮点

- 新增圆形栖声图标、应用名称和 Windows 发布包命名。
- 音乐页新增可开关的右侧歌词预览，跟随当前播放歌曲显示封面、标题、歌手和歌词。
- 艺术家、专辑和详细信息页面统一到当前播放器的玻璃拟态视觉体系。
- 左侧侧栏切换主页面时新增主内容区淡入淡出过渡。
- 修复播放器内残留中文乱码和版本信息显示错误。
- 保留本地曲库、歌单、桌面歌词、在线歌词/封面匹配、ReplayGain、CUE、快捷键和托盘控制等能力。

## 发布产物

- `Qisheng-Player-v1.0.0-Windows-x64.zip`
- `Qisheng-Player-v1.0.0-Setup-x64.exe`

## 升级提示

旧版 `coriander_player` 数据会在首次启动时迁移到 `Documents/qisheng_player`。如果你有重要曲库配置，建议升级前备份旧目录。
