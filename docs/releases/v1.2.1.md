# Qisheng Player v1.2.1

发布日期：2026-05-05

这是 `v1.2.x` 的小补丁更新，重点修复启动后静默检查到新版本但更新窗口没有弹出的问题。

## 修复

- 修复启动自动检查更新时使用了 Navigator 外层 context，导致发现新版本后只写日志、不弹出更新窗口的问题。
- 启动检查保持静默体验：没有新版本时不显示“无新版本”提示，网络异常仅写入日志。

## 验证

- 新增启动更新提示回归测试，覆盖更新提示组件位于 Router 外层时仍能弹出对话框的场景。

## 发布产物

- `Qisheng-Player-v1.2.1-Windows-x64.zip`
- `Qisheng-Player-v1.2.1-Setup-x64.exe`

## 升级提示

- `v1.1.0` 和 `v1.2.0` 已安装版本的启动弹窗可能不会出现；如未自动提示，请进入设置页手动点击“检查更新”。
- 升级到 `v1.2.1` 后，后续版本的启动自动更新提示会恢复正常。
