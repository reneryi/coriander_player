# Qisheng Player v1.2.0

发布日期：2026-05-05

本版本聚焦页面转场统一、艺术家详情共享元素动画，以及文件夹页和设置页 hover 频闪修复。`v1.1.0` 用户启动播放器后会自动检查 GitHub 最新正式 Release，发现 `v1.2.0` 后弹出更新提示并提供下载入口。

## 亮点

- 艺术家页进入详情页时，艺术家头像会平滑移动到详情页头像位置；返回时反向衔接。
- 统一页面切换时旧页面表现：旧页会同步模糊、向下位移、轻微缩放并淡出，减少突然消失的突兀感。
- 保留专辑页进入/返回详情页的封面 Hero 动画表现。

## 改进

- 将专辑页已有 Hero 导航守卫扩展为通用 artwork Hero 机制，便于艺术家头像复用。
- 文件夹页移除列表项悬浮路径 Tooltip，避免 hover 时悬浮层与玻璃背景叠加闪烁。
- 设置页分组面板改用稳定玻璃策略，降低控件 hover 时背景模糊反复重绘的概率。
- `AppSurface` 支持按场景选择背景模糊行为，复杂页面可以优先保证稳定性。

## 修复

- 修复文件夹页鼠标掠过文件夹选项时可能出现的频闪。
- 修复设置页控件 hover 时可能出现的类似频闪。
- 修复旧页面切换时只模糊、不连续淡出导致的生硬消失。

## 发布产物

- `Qisheng-Player-v1.2.0-Windows-x64.zip`
- `Qisheng-Player-v1.2.0-Setup-x64.exe`

## 升级提示

- 如使用便携版，请完整解压 zip 后再启动新版。
- 播放器暂不支持应用内自动替换安装包；更新提示会打开 GitHub Release 页面下载新版。
- 为确保 `v1.1.0` 用户收到提示，GitHub Release 请使用 tag `v1.2.0`，并保持 `prerelease` 为 `false`。
